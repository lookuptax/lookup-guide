日本是中国重要的贸易伙伴，了解日本的税务识别号码系统对于中日贸易和跨境电商至关重要。本指南详细介绍日本的两种主要税务识别号码及其用途。

## 个人编号 (My Number / マイナンバー)

个人编号（通常称为"My Number"或日语"マイナンバー"）是在日本社会保障和税务编号系统框架内分配给日本居民的12位标识符。自2015年10月起，所有在住民票中登记的居民都被分配了My Number。这些号码通过挂号信从各市、区、町、村寄送到登记住址。

**值得注意的是，持有在留卡的外国人也会被分配My Number**。今后，无论是因出生还是从海外迁入而新登记为居民的个人，都将获得My Number。

### My Number的主要特点

- **格式**：12位数字
- **分配对象**：所有日本居民（包括外国人）
- **用途**：社会保障、税务和灾害对策

### 个人编号的安全管理

12位个人编号是基于住民票代码创建的，但编号系统设计为无法从个人编号反推出住民票代码。

**关于收集个人编号的重要考虑：**
根据日本国内法和《共同报告准则》(CRS)的特定条款，位于非《多边税务行政互助公约》缔约方管辖区或与日本没有双边税收协定的报告金融机构，可以免于为CRS自动信息交换而收集日本居民的个人编号信息。

**企业经营者有责任实施重要且有效的安全管理措施**，防止个人编号的泄露、丢失或损害。这包括确保对这些12位标识符进行安全管理的适当保障和措施。

## 法人番号 (Corporate Number)

法人番号是一个由13位纯数字组成的标识符。对于根据《公司法》等法律要求登记的法人（注册法人），该编号对应于根据《商业登记法》派生的公司法人编号（12位）加上1位检查位。

**与个人编号不同，法人番号的使用范围没有限制，任何人都可以自由使用**。

### 法人番号的分配对象

法人番号分配给以下各类实体：

1. **国家机关**
2. **地方政府**
3. **已设立并登记的法人**
4. 不属于上述类别的法人，或**不支付工资的无法律人格的法人团体**

这些实体必须提交开设办事处通知表等相关材料。

此外，即使不符合上述条件，实体也可以通过满足特定要求并通知国税厅长官来获得法人番号。

**重要提示**：每个法人只获得一个唯一的法人番号，该编号不适用于法人的分支机构或办事处。

### 法人番号格式

| 项目   | 说明                         |
| ------ | ---------------------------- |
| 位数   | 13位数字                     |
| 组成   | 12位公司法人编号 + 1位检查位 |
| 公开性 | 公开信息，任何人可查询       |

## 日本税号类型汇总

| 税号类型 | 日语名称                    | 格式     | 用途          |
| -------- | --------------------------- | -------- | ------------- |
| 个人编号 | マイナンバー (My Number)    | 12位数字 | 个人税务/社保 |
| 法人番号 | 法人番号 (Corporate Number) | 13位数字 | 企业识别      |

## 对中国企业的重要性

对于与日本公司进行贸易的中国企业：

- **B2B交易**：可能需要验证日本合作伙伴的法人番号
- **跨境电商**：了解日本税务系统有助于合规运营
- **投资**：了解日本税号体系是尽职调查的重要部分
