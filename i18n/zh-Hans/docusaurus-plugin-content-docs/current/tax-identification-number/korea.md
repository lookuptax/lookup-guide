韩国是中国重要的贸易伙伴和技术来源地，了解韩国的税务识别号码系统对于中韩贸易和跨境电商至关重要。本指南详细介绍韩国的三种主要税务识别号码及其用途。

## 居民登记号 (Resident Registration Number)

韩国的居民登记号（RRN）由内务部向居住在韩国的韩国公民发放。当企业从员工工资中代扣税款，或个人为所得税、转让税、遗产税和/或赠与税申报纳税时，这个唯一标识符发挥着关键作用。

### 居民登记号格式

居民登记号由13位数字组成，格式为：**XXXXXX-XXXXXXX**

| 位置  | 说明                                        |
| ----- | ------------------------------------------- |
| 前6位 | 出生日期（例如：1985年1月2日出生 → 850102） |
| 第7位 | 性别标识                                    |
| 后6位 | 其他识别信息                                |

## 事业者登录号 (Business Registration Number / BRN)

事业者登录号由相关管辖区的地区税务局局长签发，是在韩国开展业务的实体的重要标识符。这个唯一编号作为所有税务相关申报（包括企业所得税申报）的税务识别号码（TIN）。

### 事业者登录号格式

- **位数**：10位数字
- **结构**：XXX-XX-XXXXX
- **签发机构**：地区税务局

## 法人登记号 (Corporation Registration Number / CRN)

对于已在法院登记成立的法人实体，会获得由法院管理的法人登记号。值得注意的是，在特定情况下，法人登记号也被认可并接受为该实体的税务识别号码。

### 法人登记号格式

- **位数**：13位数字
- **结构**：XXXXXX-XXXXXXX
- **管理机构**：法院

## 韩国税号类型汇总

| 税号类型     | 韩语名称       | 格式                  | 用途     |
| ------------ | -------------- | --------------------- | -------- |
| 居民登记号   | 주민등록번호   | 13位 (XXXXXX-XXXXXXX) | 个人税务 |
| 事业者登录号 | 사업자등록번호 | 10位 (XXX-XX-XXXXX)   | 企业税务 |
| 法人登记号   | 법인등록번호   | 13位 (XXXXXX-XXXXXXX) | 法人识别 |

## 对中国企业的重要性

对于与韩国公司进行贸易的中国企业：

- **B2B交易**：需要验证韩国合作伙伴的事业者登录号
- **跨境电商**：在韩国电商平台（如Coupang、Gmarket）销售需要了解韩国税务体系
- **技术合作**：与韩国科技公司合作时，了解其法人登记号有助于尽职调查
