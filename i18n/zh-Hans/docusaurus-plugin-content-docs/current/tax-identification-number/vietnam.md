越南是中国"一带一路"倡议的重要节点国家，近年来越来越多的中国企业将生产基地迁移到越南。了解越南的税务识别号码系统对于中越贸易至关重要。

## 税务识别号码 (Mã số thuế / MST)

"Mã số thuế (MST)"是发放给相关实体（包括公司、合伙企业和子公司）和个人用于税务管理的税务识别号码。组织或个人可以通过访问[越南税务总局网站](https://thuedientu.gdt.gov.vn/)在线申请MST号码。

### MST号码格式

MST号码有两种格式：**10位数字**或**13位数字**。

| 类型 | 位数     | 适用对象                     |
| ---- | -------- | ---------------------------- |
| 10位 | 10位数字 | 母公司和个人                 |
| 13位 | 13位数字 | 子公司（前10位与母公司相同） |

**示例：**

- 10位号码：<font color="green">03</font><font color="orange">1440905</font><font color="red">8</font>
- 13位号码：<font color="green">03</font><font color="orange">1440905</font><font color="red">8</font>-<font color="blue">002</font>

### MST号码结构详解

| 位置      | 数字    | 含义                                                    |
| --------- | ------- | ------------------------------------------------------- |
| 第1-2位   | 03      | 省份代码（企业注册地）。如企业日后迁址，MST号码保持不变 |
| 第3-9位   | 1440905 | 随机数字（0000001到9999999之间）                        |
| 第10位    | 8       | 校验位（根据前9位计算）                                 |
| 第11-13位 | 002     | 可选，仅用于子公司（001到999之间）                      |

## 对中国企业的重要性

对于在越南开展业务的中国企业：

- **供应商验证**：验证越南供应商的MST号码确保其合法注册
- **制造业转移**：了解越南税务体系有助于顺利设立工厂
- **跨境电商**：向越南销售产品需要了解MST相关税务要求
- **进出口贸易**：MST号码是海关清关和税务申报的必备信息
