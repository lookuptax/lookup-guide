印度尼西亚是东南亚最大的经济体，拥有超过2.7亿人口的巨大市场。了解印尼的税务识别号码系统对于中国企业开拓印尼市场至关重要。

## 纳税人识别号 (Nomor Pokok Wajib Pajak / NPWP)

印尼的税务识别号码称为**Nomor Pokok Wajib Pajak (NPWP)**，包含以下信息：

**A. NPWP的作用**：作为纳税人的唯一标识符，用于识别和确认纳税人在税务领域的权利和义务。

**B. 发放对象**：符合税法规定的主观和客观条件的纳税人将获得NPWP。

**C. 2022年7月14日起的新规定：**

1. 印尼居民个人纳税人必须使用其国民身份号码（Nomor Induk Kependudukan/NIK）
2. 非印尼居民个人纳税人、企业纳税人和政府机构纳税人必须使用税务总局签发的16位NPWP

### NPWP登记要求

个人和实体在满足《税务程序法》(GPTPL) 第2条第1款规定的主观和客观条件后，需要进行纳税人登记。

特殊情况：

- 已婚女性通常使用丈夫的NPWP进行税务活动
- 在特定条件下，可以选择单独征税（如法院判决分居、书面申请收入和资产分离协议等）
- 税务总局长可依职权为符合条件但未自行登记的纳税人签发NPWP

### NPWP格式

#### 印尼居民个人纳税人

| 项目     | 说明                           |
| -------- | ------------------------------ |
| 位数     | 16位数字（2022年7月14日起）    |
| 对应关系 | 与国民身份号码（NIK）相同      |
| 旧格式   | 15位格式有效期至2023年12月31日 |

## 印尼税号类型汇总

| 税号类型      | 位数 | 适用对象                  |
| ------------- | ---- | ------------------------- |
| NPWP (新格式) | 16位 | 所有纳税人（2022年7月起） |
| NIK           | 16位 | 印尼居民个人              |
| NITKU         | 变动 | 分支机构                  |
| NPWP (旧格式) | 15位 | 过渡期内有效              |

## 对中国企业的重要性

对于进入印尼市场的中国企业：

- **最大市场**：印尼是东南亚最大的经济体和人口最多的国家
- **跨境电商**：在Tokopedia、Shopee Indonesia、Lazada Indonesia等平台销售需要了解NPWP体系
- **投资设厂**：在印尼设立公司需要注册NPWP
- **供应商验证**：验证印尼供应商的NPWP确保合法合规
- **数字经济**：印尼是东南亚数字经济增长最快的市场
