Die Taxpayer Identification Number (TIN) umfasst verschiedene Arten von Steuer-IDs, von denen die wichtigsten die Social Security Number (SSN), die Employer Identification Number (EIN) und die Individual Taxpayer Identification Number (ITIN) sind. Daneben gibt es noch die Taxpayer Identification Number for Pending U.S. Adoptions (ATIN) und die Preparer Taxpayer Identification Number (PTIN).

## Social Security Number – SSN

Die Sozialversicherungsnummer (Social Security Number, SSN) wird von der Social Security Administration (SSA) ausgestellt. Ihr Zweck ist es, Konten (Löhne / Einkommen) von Einzelpersonen im Rahmen des Sozialversicherungsprogramms zu verfolgen, Sozialversicherungsleistungen zu verwalten und für andere Identifikationszwecke zu dienen. Sie wird an US-Bürger, ständige Einwohner und arbeitende oder befristete Einwohner ausgegeben.

### Format der SSN
Die SSN ist eine 9-stellige eindeutige Nummer.

<font color="green">123</font>-<font color="orange">45</font>-<font color="red">6789</font> 

#### Aufbau der SSN

Die SSN ist in 3 Teile unterteilt: Area Code, Group Number, Serial Number.

<font color="green">123</font> — Die ersten 3 Ziffern stellen den Area Code dar. Der Area Code wurde ursprünglich basierend auf der Postleitzahl der Postanschrift im Antrag zugewiesen. Die Postanschrift des Antragstellers muss nicht mit seinem Wohnort übereinstimmen. Wie auf der <a href="https://www.ssa.gov/history/ssn/geocard.html" target="_blank">offiziellen Website der SSA</a> erwähnt, sollte man dem "geografischen Code" jedoch nicht zu viel Bedeutung beimessen.

<font color="orange">45</font> — Die mittleren 2 Ziffern sind die Group Number. Sie reichen von 01 bis 99.

<font color="red">6789</font> — Die letzten vier Ziffern (Serial Number) werden fortlaufend von 0001 bis 9999 vergeben.

## Individual Taxpayer Identification Number – ITIN

Die Individual Taxpayer Identification Number (ITIN) wird vom Internal Revenue Service (IRS) an Personen ausgestellt, die eine Steuererklärung abgeben müssen, aber keine US-Bürger sind und keine Social Security Number haben oder dafür nicht berechtigt sind. ITINs werden unabhängig vom Einwanderungsstatus ausgestellt, da sowohl ansässige als auch nicht ansässige Ausländer eine US-Steuererklärungs- oder Meldepflicht haben können.

### Format der ITIN
Die ITIN ist eine 9-stellige eindeutige Nummer.

<font color="green">912</font>-<font color="orange">53</font>-<font color="red">6789</font>

#### Aufbau der ITIN

Eine ITIN beginnt immer mit einer 9.

9xx-xx-xxxx

Die 4. und 5. Ziffern liegen in bestimmten Bereichen (z.B. 50 bis 65, 70 bis 88, 90 bis 92 und 94 bis 99).

## Employer Identification Number - EIN

Die EIN wird vom IRS an Einzelunternehmer, Personengesellschaften, Kapitalgesellschaften (einschließlich LLCs) und andere Einheiten für Steuererklärungs- und Meldezwecke ausgestellt. Für deutsche Unternehmen, die in den USA tätig sind, ist dies oft die wichtigste Nummer.

### Format der EIN
Die EIN ist eine 9-stellige eindeutige Nummer.

<font color="green">123</font>-<font color="orange">45</font>-<font color="red">6789</font>

#### Aufbau der EIN

XXX-XX-XXXX wobei X eine beliebige Ziffer zwischen 0-9 sein kann.
