Das **VIES** (VAT Information Exchange System), in Deutschland auch als **MIAS** (Mehrwertsteuer-Informationsaustauschsystem) bekannt, ist eine Suchmaschine der Europäischen Kommission (Generaldirektion Steuern und Zollunion). Sie ermöglicht es Unternehmen, die Gültigkeit einer Umsatzsteuer-Identifikationsnummer (USt-IdNr) von Geschäftspartnern zu überprüfen, die in der Europäischen Union registriert sind.

## Schritte zur Verwendung des VIES/MIAS-Portals

Die Europäische Kommission bietet einen kostenlosen Dienst zur Validierung von USt-IdNrn an.

1.  Besuchen Sie die offizielle [VIES-Website](https://ec.europa.eu/taxation_customs/vies/).
2.  Wählen Sie den **Mitgliedstaat** aus dem Dropdown-Menü (das Land, das die Nummer ausgestellt hat).
3.  Geben Sie die **Umsatzsteuer-Nummer** (ohne Ländercode) ein.
4.  Klicken Sie auf **Verifizieren**.

### Mögliche Ergebnisse

- **Gültig (Ja):** Die Nummer ist im System registriert und derzeit gültig. Das System zeigt oft auch den Namen und die Adresse des Unternehmens an (abhängig von den Datenschutzgesetzen des jeweiligen Landes).
  - _Tipp:_ Drucken Sie das Ergebnis als Nachweis für Ihre Buchhaltung ("qualifizierte Bestätigung").
- **Ungültig (Nein):** Die Nummer ist nicht registriert oder ungültig.
  - _Achtung:_ Führen Sie keine steuerfreien innergemeinschaftlichen Lieferungen an diesen Partner durch, bis die Nummer geklärt ist.
