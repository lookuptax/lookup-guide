## Verificación del DNI en el sitio web de RENAPER

La opción de búsqueda del DNI (Documento Nacional de Identidad) está disponible de forma gratuita en el sitio web de RENAPER Argentina. RENAPER (Registro Nacional de las Personas) es la autoridad argentina encargada del registro civil nacional y la emisión de documentos de identidad. Puedes acceder a la consulta de DNI [aquí](https://tramites.renaper.gob.ar/mi_ejemplar/). Esta búsqueda requiere agregar la fecha de nacimiento y el sexo para validar el DNI exitosamente.

- Estado
- Fecha de emisión
