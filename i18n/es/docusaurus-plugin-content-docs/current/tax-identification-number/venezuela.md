## RIF - Registro de Información Fiscal

El **RIF (Registro de Información Fiscal)** es el número de identificación tributaria que el SENIAT (Servicio Nacional Integrado de Administración Aduanera y Tributaria) asigna en Venezuela a personas naturales y jurídicas para la administración de impuestos.

### ¿Quiénes deben tener RIF?

Están obligados a obtener RIF:

- Personas naturales que realicen actividades económicas
- Personas jurídicas (empresas, sociedades, asociaciones)
- Sucursales y establecimientos permanentes de empresas extranjeras
- Entidades públicas y privadas con obligaciones tributarias

### Formato del RIF

El RIF tiene **10 caracteres** con el siguiente formato:

**Formato:** `X-NNNNNNNN-N`

| Componente        | Descripción                             |
| ----------------- | --------------------------------------- |
| **Primera letra** | Tipo de contribuyente (V, E, J, P, G)   |
| **Guion**         | Separador                               |
| **8 dígitos**     | Número secuencial (00000001 - 99999999) |
| **Guion**         | Separador                               |
| **Último dígito** | Dígito verificador (0-9)                |

#### Letras de tipo de contribuyente

| Letra | Tipo de contribuyente             |
| ----- | --------------------------------- |
| **V** | Venezolano (persona natural)      |
| **E** | Extranjero (persona natural)      |
| **J** | Persona jurídica (empresa)        |
| **P** | Pasaporte (extranjero sin cédula) |
| **G** | Gobierno (entidades públicas)     |

### Ejemplos de RIF válidos

| Tipo                       | Ejemplo        |
| -------------------------- | -------------- |
| Persona natural venezolana | `V-12345678-0` |
| Persona natural extranjera | `E-87654321-5` |
| Empresa (persona jurídica) | `J-30123456-7` |
| Entidad gubernamental      | `G-20000001-3` |

### Desglose del RIF

Tomando el ejemplo `V-12345678-0`:

| Parte            | Valor    | Significado                  |
| ---------------- | -------- | ---------------------------- |
| 1° carácter      | V        | Venezolano (persona natural) |
| 2°-9° caracteres | 12345678 | Número asignado por SENIAT   |
| 10° carácter     | 0        | Dígito verificador calculado |

> **Nota:** Los guiones son obligatorios y forman parte del formato oficial del RIF.
