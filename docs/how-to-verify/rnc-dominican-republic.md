## RNC check on  Impuestos Internos website

RNC number search option is available for free on the Dominican Republic Internal Revenue (Impuestos Internos) website. You can access the RNC search [here](https://dgii.gov.do/herramientas/consultas/Paginas/RNC.aspx). This search allows you to search for a company by its RNC or c√©dula(ID). This tools also provides an option to search the companies by its name. This search can also be used to search for individuals.
